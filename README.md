# hello-world

A simple tip calculator CLI built in Python.

## Usage

```bash
python main.py
```

Example:

```
=== Tip Calculator ===
Bill amount: $80
Tip percentage: 18
Number of people: 2

Tip:        $14.40
Total:      $94.40
Per person: $47.20
```

## Run tests

```bash
python -m pytest test_main.py
```
